# Cyrus's Scoop Bucket

A Scoop bucket for my open source projects.

## Installation

Add this bucket to Scoop:

```powershell
scoop bucket add jcyrus https://github.com/jcyrus/scoop-bucket
```

## Available Apps

### [GhostWire](https://github.com/jcyrus/GhostWire)

Ephemeral TUI chat client with E2EE.

```powershell
scoop install ghostwire
```

### [Spektr](https://github.com/jcyrus/spektr)

Blazing-fast TUI utility for cleaning development artifacts.

```powershell
scoop install spektr
```

### [NebulaVault](https://github.com/jcyrus/nebulavault)

High-performance, secure SSH connection manager with GPU-accelerated GUI.

```powershell
scoop install nebulavault
```

### [BrowserPort](https://github.com/jcyrus/browserport)

Cross-platform browser picker utility.

```powershell
scoop install browserport
```
